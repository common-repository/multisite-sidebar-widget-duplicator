Create a duplicate of all widgets in master site for each newly created sub-site.

== Description ==

Plugin works automatically. When activated, it duplicates all the widgets to new Wordpress websites in your multisite network.

== Installation ==

1. Upload `multisite-sidebar-widget-duplicator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

`Does it copy widgets to already created sub-sites?`

No, plugin works just when creating new sub-site.
